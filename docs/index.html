How It Works

From slash command to commit

Four simple steps ‚Äî the agent handles the rest.

/task

You send a task in Discord. The bot creates a thread and provisions a git worktree for this session.

Agent works

Copilot agent works autonomously. Every tool call passes through the deny-by-default policy engine.

Live stream

Output streams in real-time into the thread. Secrets are redacted before posting. You can send follow-ups.

Push gate

When it's time to push, the agent asks for approval via buttons. You review the diff and decide.

Quick Start

Prerequisites: Git, Node.js ‚â• 18, npm, and Copilot CLI (authenticated).

Bot Setup

Create your Discord bot

Five steps to get your bot running in a Discord server.

1

Create application

Go to the Discord Developer Portal ‚Üí New Application ‚Üí Bot ‚Üí copy the token.

2

Enable Message Content Intent

Under Bot settings, enable the Message Content Intent toggle.

3

Set permissions

Send Messages, Embed Links, Attach Files, Use Slash Commands, Create Public Threads, Send Messages in Threads

4

Generate invite URL

OAuth2 ‚Üí URL Generator ‚Üí scopes: bot, applications.commands ‚Üí invite to your server.

5

Run the agent

Set DISCORD_TOKEN and run ./agent.sh (Linux/macOS) or .\agent.ps1 (Windows).
