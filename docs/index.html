How It Works

From message to commit

Four simple steps ‚Äî the agent handles the rest.

Message

DM the bot or @mention it in a channel. A session and git worktree are auto-provisioned.

Agent works

Copilot agent works autonomously. Every tool call passes through the deny-by-default policy engine.

Live stream

Output streams in real-time. Secrets are redacted before posting. Reply in the thread for follow-ups.

Approval gates

git push and outside-workspace access trigger button prompts. You review and decide.

Interaction

Message-first + 7 admin commands

Just chat ‚Äî DM, @mention, or reply in a thread. Admin slash commands are available as escape hatches.

Method Description

"stop"Type stop in a DM or thread to abort the running task.

Bot Setup

Create your Discord bot

Five steps to get your bot running in a Discord server.

1

Create application

Go to the Discord Developer Portal ‚Üí New Application ‚Üí Bot ‚Üí copy the token.

2

Enable Message Content Intent

Under Bot settings, enable the Message Content Intent toggle.

3

Set permissions

Send Messages, Embed Links, Attach Files, Read Message History, Use Slash Commands, Create Public Threads, Send Messages in Threads

4

Generate invite URL

OAuth2 ‚Üí URL Generator ‚Üí scopes: bot, applications.commands ‚Üí invite to your server.

5

Run the agent

Set DISCORD_TOKEN and run ./agent.sh (Linux/macOS) or .\agent.ps1 (Windows).
